## [0.1.0] - Unreleased

### Added

- Modular simulation, Three.js, physics, character, combat, bots, UI, debug, and localization packages.
- Physical projectiles, grenades, destructible props, ragdolls, model loading, and component-driven animation.
- Game build CLI with Web archive and Tauri 2 desktop export support.
- Deterministic tests, AI modification tasks, and three private browser examples.
